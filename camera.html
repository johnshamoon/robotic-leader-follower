camera¶

camera

Author: Steven Dropiewski

camera.ABSOLUTE = 1¶

The value set to identify absolute degrees.

camera.RELATIVE = 0¶

The value set to identify relative degrees.

class camera.Camera(bus_number=1, pan_channel=1, tilt_channel=2, camera_type=1)¶

Creates a camera object that can be panned and tilted.

The camera can pan in [0, 180] degrees and can tilt in [0, 150] degrees. The camera operates in absolute degrees by default, but can be specified to work in relative degrees during object creation.

Parameters:
  • bus_number (int) – The bus number of the servos.
  • pan_channel (int) – Pan servo’s I2C channel.
  • tilt_channel (int) – Tilt servo’s I2C channel.
  • camera_type (int) – Camera.ABSOLUTE or Camera.RELATIVE angles.

pan(degree)¶

Pans the camera.

Controls the horizontal movement of the camera in [0, 180] degrees. 0 refers to the vehicle’s right while 180 refers to the vehicle’s left. The degrees will be clamped to [0, 180].

Parameters:degree (int) – The number of degrees.
calibrate_pan(degree)¶

Calibrates the camera’s panning servo.

Writes a new pan offset value to the config file and changes the current pan offset. The calibration value can be negative.

Parameters:degree (int) – The number of degrees.
get_pan_angle()¶

Gets the current pan angle of the camera.

Returns:The current pan angle (in degrees).
Return type:int
tilt(degree)¶

Tilts the camera.

Controls the vertical movement of the camera in [0, 150] degrees. 0 is directly in front of the car, 90 faces upwards, and 150 faces the Raspberry Pi. The degrees will be clamped to [0, 150].

Parameters:degree (int) – The number of degrees.
calibrate_tilt(degree)¶

Calibrates the camera’s tilting servo.

Writes a new tilt offset value to the config file and changes the current tilt offset. The calibration value can be negative.

Parameters:degree (int) – The number of degrees.
get_tilt_angle()¶

Gets the current tilt angle of the camera.

Returns:The current tilt angle (in degrees).
Return type:int
reset_camera()¶

Resets the camera to default position.

Sets the camera pan to 90 and the tilt to 0.

camera.main()¶

Demonstrates the camera’s panning and tilting functionality.

Creates a camera object using absolute degrees and pans from 0 to 180, then from 180 to 0. The camera then tilts from 0 to 150, then 150 to 0. Then creates a camera object using relavtive degrees and pans from 0 to 180, then from 180 to 0. The camera then tilts from 0 to 150, then 150 to 0. camera_delay is used to allow smooth camera movement in small distances. camera_delay_long is used to allow the camera to move long distances fully.
