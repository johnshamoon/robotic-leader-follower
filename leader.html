class leader.Leader(test_mode=False)¶

Leader vehicle class.

Takes input from /dev/input/js0 and directs the vehicle. To turn left or right, use the d-pad. To drive forward, use the right trigger. To drive backward, use the left trigger.

Parameters:test_mode (boolean) – Test mode. Disables movements (off by default).
STRAIGHT_ANGLE = 90¶

The angle that the hardware associates as straight.

set_speed(position)¶

Sets the speed of the vehicle based on controller input.

If the position is -1, the speed will be 0. If position is between (-1,1), the speed will increase as the position goes to 1. If the position is 1, the speed will be LEADER.MAX_SPEED.

Parameters:position (float) – The value of the button input.
drive()¶

Moves the vehicle forward.

The vehicle moves forward at the last set speed.

reverse()¶

Moves the vehicle backward.

The vehicle moves backward at the last set speed.

turn(position)¶

Turns the wheels based on controller input.

If position is -1, the wheels turn left. If position is 1, the wheels turn right. Otherwise, the wheels turn straight.

Parameters:position (float) – The value returned from InputController.get_input().
Returns:The turn angle.
Return type:float
Returns:None if position is not a number.
Return type:None
lead()¶

Controls a leader vehicle.

Moves the leader vehicle based on controller input.

leader.main()¶

Instantiates a Leader object and continuously calls Leader.lead().
