class inputcontroller.InputController(debug=False)¶

Module used to get input from an Xbox One S controller.

Assumes the device is /dev/input/js0.

Recognized Buttons:
  1. Left Stick X and Y
  2. Right Stick Z and Rotation Z
  3. Left and Right Triggers
  4. Left and Right Buttons
  5. D-Pad Up, Left, Down, and Right
  6. A, B, X, and Y
  7. Start/Select Buttons
  8. Left and Right Stick Buttons
Parameters:debug (bool) – Enabling debug causes get_input() to print values.

get_input()¶

Takes input from /dev/input/js0 and returns a pair of (code, value).

If a button is being pressed, value is 1. When the button stops being pressed, state is 0. If a button is not being pressed, the code will be -1.

If an axis is being used, the value will be in [-1, 1]. When the input on the axis stops, the code is -1 and the position is -2.

Returns:A code and a value representing the linux button/axis code and its value.
Return type:string, float
Returns:A code when the input button/axis is not being used anymore.
Return type:int, int
inputcontroller.main()¶

Instantiates an InputController object with debugging enabled and continuously calls InputController.get_input().
