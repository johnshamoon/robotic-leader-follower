class tagrec.TagRecognition(resolution=90, dead_zone=1.45, marker_length=0.06, contrast=1, brightness=0)¶

Tag Recognition class.

Takes input from the camera to look for an ARTag and returns useful information about the tag.

Parameters:
  • resolution (int) – The camera’s resolution.
  • dead_zone (float) – The dead zone to consider everything to be directly in front of the camera.
  • marker_length (float) – The square length of the ARTag in meters.
  • contrast (float) – The contrast multiplier. A value in (0, 1) lowers the contrast and a value in [1, 100] increases the contrast.
  • brightness (int) – The brightness value in (-127, 127).
RESOLUTIONS = {90: [160, 90], 144: [176, 144], 180: [320, 180], 240: [426, 240], 360: [480, 360], 480: [720, 480], 720: [1280, 720], 1080: [1920, 1080]}¶
